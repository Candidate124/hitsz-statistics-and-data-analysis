# hitsz-statistics-and-data-analysis
## 概率和数理统计实验
随课程进度完成的实验报告。
### src文件夹用于存放程序代码，运行其中的.m文件。
### export文件夹中的.mlx文件仅为了将其导出为word，方便写实验报告时优雅地粘贴源代码到word文档中；对应的word文档也在export中显示。
  :bangbang:<mark>**_如果想要运行代码，一定要确保打开src文件夹中的.m文件，并且确认MATLAB左侧面板的路径为...Experiment_1\src。_**</mark>
  
  :bangbang:<mark>**_若路径中包含export文件夹或其中的.mlx文件，将会导致.m文件无法运行。_**</mark>
  
  若出现报错如“MATLAB无法运行该文件，因为MATLAB路径中存在...Experiment_1\Task1.mlx，导致该文件被遮蔽。”
  大概率按上述操作即可正常运行src文件中的.m文件。
