# hitsz-statistics-and-data-analysis
## 概率和数理统计实验
随课程进度完成的实验报告。
### 总体分布
- src文件夹用于存放程序代码，运行其中的.m文件。
- export文件夹中的.mlx文件是为了将其导出为word，方便写实验报告时优雅地粘贴源代码到word文档中；对应的word文档也在export中显示。
  ==如果想要运行代码，一定要确保在MATLAB中打开src文件夹中的.m文件，并且确认MATLAB左侧面板的路径为Experiment_1\src。==
  ==若路径中包含export文件夹或其中的.mlx文件，将会导致.m文件无法运行。==
  ![报错如图]()
  出现如图报错大概率按上述方法即可正常运行src中的.m文件。
